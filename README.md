# 🧮 Calculadora Interativa em Python

Este é um projeto prático que desenvolvi para consolidar meus conhecimentos fundamentais em Python. O foco principal foi criar uma ferramenta funcional que trate erros comuns de entrada do usuário.

### 🚀 O que este projeto demonstra:
* **Lógica de Programação:** Uso de estruturas condicionais (`if/elif/else`) e laços de repetição (`while`).
* **Tratamento de Exceções:** Implementação de `try/except` para evitar que o programa feche caso o usuário digite um valor não numérico.
* **Validação de Dados:** Verificação lógica para impedir a divisão por zero.
* **Interatividade:** Menu amigável via terminal com opção de saída controlada pelo usuário.

### 🛠️ Tecnologias:
* Python 3.x

### 💻 Como rodar:
1. Clone o repositório.
2. No terminal, execute: `python calculadora.py`
